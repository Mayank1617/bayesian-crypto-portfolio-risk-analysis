# Bayesian Cryptocurrency Portfolio Risk Analysis using MCMC

## Overview
This project develops a Bayesian risk analysis framework for cryptocurrency portfolios using Markov Chain Monte Carlo (MCMC) methods. Unlike traditional approaches that estimate a single volatility value, Bayesian inference models volatility as a probability distribution, enabling uncertainty-aware risk assessment and credible interval estimation.

## Key Features
- Bayesian volatility estimation using PyMC
- NUTS (No-U-Turn Sampler) based MCMC inference
- Posterior volatility distributions and uncertainty quantification
- Multi-asset portfolio risk analysis
- Real-world cryptocurrency market data obtained via Yahoo Finance

## Technologies Used
Python, PyMC, NumPy, Pandas, SciPy, ArviZ, Matplotlib, yfinance
